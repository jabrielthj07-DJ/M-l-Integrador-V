# Proyecto Integrador V – Veterinaria M&L

## Descripción

Este proyecto corresponde al **Integrador V** y es la continuación del sistema desarrollado para la **Veterinaria M&L**.

En el semestre anterior (**Integrador IV**) se desarrolló una **API REST** encargada de gestionar la lógica del sistema y el manejo de datos.

En este **Integrador V**, el objetivo principal es desarrollar el **Frontend del sistema** utilizando **HTML y CSS**, creando la interfaz visual que permitirá interactuar con la API desarrollada anteriormente.

## Tecnologías

* HTML
* CSS
* API REST

## Estado del Proyecto

Proyecto **actualmente en proceso de desarrollo**.
